Core Setup

What is Kali Linux in the lesson?

An open-source security package with tools arranged by category.

Think: a hacker toolkit sorted into labeled drawers.

What three ways can Kali be used?

As a direct operating system install, a virtual machine, or a live boot CD/USB.

Install it, virtualize it, or carry it.

Why does the lesson recommend VirtualBox?

It is presented as a safer testing environment for unknown packages or code, while letting Kali run beside your main OS.

VirtualBox = the sandbox where experiments happen.

What four-step chain should you memorize for the lab?

VirtualBox → Kali Linux → Update Kali → Add Metasploitable.

Build the arena before the boss fight.

What are the default Kali credentials in the notes?

Username: root | Password: toor.

Mirror memory trick: root ↔ toor.

What are the default Metasploitable credentials?

Username: msfadmin | Password: msfadmin.

The lab target is being unusually cooperative.

Update Trio

What does `apt-get update` do?

Updates the package index so Kali knows the latest available package information.

Update = check the menu.

What does `apt-get upgrade` do?

Upgrades installed packages to their latest versions.

Upgrade = order the food.

What does `apt-get dist-upgrade` do in the notes?

Performs a fuller system upgrade including dependency changes.

Dist-upgrade = renovate the whole restaurant.

What should you type when the upgrade asks to continue?

Type Y and press Enter.

The dramatic single-letter yes.

Recon Essentials

Nmap vs. Zenmap: what is the difference?

They are practically the same tool, but Nmap is command line while Zenmap is GUI.

Nmap = the brain; Zenmap = the face.

What does `nmap -O` do?

Performs OS type/version detection.

O = OS detective.

What does `nmap -p 1-65535 -T4` do?

Scans all TCP ports using the timing/speed setting shown in the lesson.

All ports, faster pace.

What is a stealth scan?

A SYN or half-open scan that does not complete the full TCP three-way handshake.

Knock, listen, leave.

What does Searchsploit do?

Searches the Exploit Database archive directly from the command line.

A search engine in a hacker hoodie.

What does Metagoofil appear to do in the screenshot?

Search online for PDF files from a target domain with a result limit.

The metadata goblin rummaging through public docs.

Command Atlas

These are the command groups added from KALI-COMMANDS.pdf, turned into a quick-scroll reference for review before the quiz.

Information Gathering

  • nmapDiscovers hosts, open ports, services, and OS versions.
  • lbdDetects whether a site uses load balancing.
  • hping3Advanced packet crafting tool using TCP, UDP, ICMP, and RAW-IP.
  • dnsenumRetrieves MX, A, and related DNS records.
  • dnsmapFinds subdomains and related domain-linked details.
  • dnstracerDetermines where a DNS server gets information for a hostname.

Vulnerability / Web / Exploitation

  • cisco-torchScans Cisco devices for vulnerabilities.
  • CATCisco Auditing Tool for common security weaknesses.
  • cge.plCisco Global Exploiter tool.
  • bedTests services for buffer overflow vulnerabilities.
  • sqlmapAutomates SQL injection attacks.
  • wpscanScans WordPress websites for vulnerabilities.
  • joomscanFinds vulnerabilities in Joomla CMS.
  • tlssledEvaluates SSL/TLS security configuration.
  • msfconsoleLaunches the Metasploit Framework.
  • msfupdateUpdates Metasploit with latest exploits.

Forensics / Sniffing / Passwords / Access

  • p0fPassive OS fingerprinting tool.
  • pdf-parserAnalyzes suspicious PDFs for embedded scripts.
  • dd_rescueRecovers data from damaged disks.
  • mitmproxyMan-in-the-middle proxy for web traffic inspection.
  • sslstripDowngrades HTTPS to HTTP to capture credentials.
  • wiresharkPacket analyzer for monitoring network traffic.
  • hydraPerforms brute-force login attacks.
  • johnPassword cracking tool using dictionary attacks.
  • unshadowCombines password files for cracking.
  • rcrackUses rainbow tables to crack hashes.
  • hash-identifierIdentifies hash types.
  • python -m SimpleHTTPServerCreates a quick web server to transfer files.
  • sbdCreates encrypted remote command connections.
  • weevelyGenerates PHP webshells for remote control.
  • cryptcatEncrypted version of Netcat for secure connections.
  • httptunnelCreates tunnels through HTTP to bypass firewalls.
  • dns2tcpSends TCP traffic through a DNS channel.
